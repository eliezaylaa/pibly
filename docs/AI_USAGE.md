# AI Usage Declaration — Pibly

## Overview

This document declares the use of Artificial Intelligence tools during the development of Pibly, in compliance with Epitech academic integrity requirements.

## What AI Did (~30%)

- Generated boilerplate code for controllers, routes, and React components
- Assisted with debugging specific errors
- Helped write unit test structure
- Generated documentation drafts
- Suggested security best practices

## What the Developer Did (~70%)

- Defined the entire product concept and business model
- Made all architecture decisions (database schema, API design, tech stack)
- Reviewed, validated and adapted all AI-generated code
- Debugged and fixed integration issues between all components
- Designed all user flows and user experience
- Made all product decisions (no subscriptions, 10% fee, session flow, video call)
- Manually tested all features end-to-end
- Deployed the application to production on Render
- Set up Docker and CI/CD pipeline
- Integrated Stripe, Socket.io, Jitsi Meet
